ğŸ“ How to Add Images:

  1. Prepare your images: Create two versions of each image:
    • Thumbnail: 400x400px (square) or 400x600px (portrait) - JPG quality 85%
    • Full size: 1200-2000px width (maintain aspect ratio) - JPG quality 90%
  2. Add files to folders: Place thumbnail in thumbnails/ folder and full image in full/ folder of the appropriate category
  3. Update JSON file: Open portfolio/data/images.json and add a new entry with image details:
    {
      "id": "category-002",
      "title": "Your Project Title",
      "description": "Project description",
      "thumbnail": "../assets/images/portfolio/category/thumbnails/filename.jpg",
      "full": "../assets/images/portfolio/category/full/filename.jpg",
      "category": "Category Name",
      "year": "2024",
      "client": "Client Name"
    }
  4. Save and test: Save the JSON file and refresh your website - images will automatically appear!
